# smart-home-control-hub-pic16f877a
Smart Home Control Hub using PIC16F877A with Bluetooth control, OLED dashboard, sensors, relay automation, EEPROM, and emergency interrupt handling.

## Features
- Bluetooth appliance control
- OLED live dashboard
- Motion detection
- Flame/gas detection
- Relay automation
- EEPROM storage

## Hardware
- PIC16F877A
- HC-05
- OLED SSD1306
- MQ-2
- LM35
- PIR
- Relay Module

## Communication Protocols
- UART
- I2C
- ADC
- Interrupts
